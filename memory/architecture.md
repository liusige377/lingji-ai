## 二、技术架构

### 整体架构
```
用户浏览器
    ↓
前端（Vercel免费托管，HTTPS自动）
    ↓ HTTP API
后端ECS服务器（101.133.136.55:3000）
    ├── 31个AI模型（GLM/Qwen/DeepSeek等）
    ├── MongoDB数据库（用户/配额/充值）
    ├── SMTP邮件服务（163邮箱）
    └── 微信公众号推送
```

### 后端（已有，零改动）
- **地址：** http://101.133.136.55:3000
- **API端点：**
  - `POST /v1/chat` — AI对话
  - `POST /v1/auth/register` — 注册
  - `POST /v1/auth/login` — 登录
  - `POST /v1/auth/reset-password` — 重置密码
  - `GET /v1/user/info` — 用户信息/余额
  - `POST /v1/user/recharge` — 充值
- **CORS：** 已配置，允许所有来源
- **状态：** 运行中 ✅

### 前端（需全新开发）
- **部署：** Vercel（免费，全球CDN，自动HTTPS）
- **技术：** 纯HTML + CSS + JavaScript（无框架，快速开发）
- **文件结构：**
  ```
  lingji-ai/
  ├── index.html          # 主入口（含所有功能模块，单文件）
  ├── README.md           # 项目说明
  └── vercel.json        # 部署配置（可选）
  ```
- **无需域名：** 用Vercel免费子域名（lingji-ai.vercel.app）
- **后期可绑域名：** 主人在Vercel后台一键绑定（可选）

### 模型策略
- **默认模型：** GLM-4-Flash（中文强·永久免费·智谱）
- **高速模型：** GLM-Z1-Flash（实测94 tok/s，最快）
- **备用：** Qwen-Plus、DeepSeek、Llama等（按需切换）
- **前端UI：** 用户可手动选模型，界面显示速度参考

## 三、功能模块规划

### 第一期（必须上线，v1.0）

#### 1. AI智能聊天 🤖
- 31个模型可选（下拉菜单）
- 对话历史（前端localStorage）
- 模型切换不影响上下文
- 发送按钮 + Enter快捷键
- 余额显示（登录后）
- **入口：** 首页主按钮 + 工具卡

#### 2. AI角色扮演 🎭
- 角色列表：鲁迅、李白、乔布斯、爱因斯坦、林肯、林子容、心理医生、商业顾问
- 每个角色有独特System Prompt
- 对话历史可延续
- **入口：** 首页工具卡

#### 3. AI写作工坊 ✍️
- 5种模式切换（单页面无刷新切换）：
  - 📝 文章写作（主题+类型+字数+补充要求）
  - 📣 营销文案（产品+用户画像）
  - 📧 邮件撰写（场景+内容）
  - 📋 内容总结（粘贴内容+要点数量）
  - 🌐 中英翻译（内容+风格）
- 输出结果显示在下半区
- **入口：** 首页工具卡

#### 4. AI起名大师 🏆
- 3种场景切换：
  - 👶 宝宝取名（姓氏+性别+出生年月+期望）
  - 🏢 公司取名（行业+核心理念+要求）
  - 📛 品牌取名（类型+定位+目标用户）
- AI生成5个好名字，每个附解释
- **入口：** 首页工具卡

#### 5. 快捷聊天入口（进AI聊天，自动填问题）💕
- 💕 土味情话 → AI聊天填"写10句土味情话"
- 🔮 塔罗占卜 → AI聊天填"塔罗占卜，洗牌抽3张牌"
- 🌙 周公解梦 → AI聊天填"梦到蛇追我，周公解梦"
- 🏮 对联藏头诗 → AI聊天填"写一副春开头福结尾的对联"
- 💼 简历面试 → AI聊天填"面试产品经理，5个问题和答案"
- 📚 学习私教 → AI聊天填"讲解勾股定理并出3道题"

### 第二期（后续迭代）

#### 6. AI塔罗占卜 🔮
- 用户输入问题 → AI模拟洗牌抽3张牌 → AI解读每张牌含义+综合分析
- 纯聊天实现，不需要真实卡牌图片

#### 7. 周公解梦 🌙
- 用户描述梦境 → AI按周公解梦传统解读吉凶
- 梦境关键词识别 → 对应解读

#### 8. 简历优化 💼
- 用户粘贴简历 → AI优化措辞、结构、关键词
- 提供面试官视角的改进建议

#### 9. 学习私教 📚
- 拍题解题（用户描述题目）
- 知识讲解（指定知识点）
- 练习题生成

#### 10. 对联/藏头诗 🏮
- 对联：指定上联/下联任意一个字开头
- 藏头诗：输入主题，生成藏头诗

## 四、用户系统（复用后端）

### 登录/注册流程
- 用户名 + 邮箱 + 密码 + 邮箱验证码 → 注册
- 用户名 + 密码 → 登录
- 邮箱验证码 → 重置密码

### 配额系统（复用现有）
- 新用户赠送初始配额
- 付费充值增加配额
- 显示余额，余额不足提示充值

### 游客模式
- 不登录也能用AI聊天（用公共配额池或限制调用量）
- 登录后享受完整功能和配额
